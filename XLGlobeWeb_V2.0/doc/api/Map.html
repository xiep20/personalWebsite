Globe

Globe对象用于构建应用程序的基本小控件。它将所有标准的cesium部件组合到一个可重用的包中。可以通过使用mixins来扩展小控件，它为各种应用程序添加了有用的功能。

▸ viewer.centerAt(centeropt, options)

定位。

Parameters
centeropt (Object) 可选参数，如不配置将定位到默认范围，配置参数为xmin,ymin,xmax,ymax分别指定的是x y坐标矩阵范围。
