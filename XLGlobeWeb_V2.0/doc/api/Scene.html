Scene

Scene是XLGlobe虚拟场景中所有3D图形对象和状态的容器。它公开允许您以编程方式更改场景的属性，并在用户与其交互时激发事件。

Instance Members

BookmarkWidget(viewer)

BookmarkWidget是XLGlobe虚拟场景中书签列表控件，它公开允许您以书签的方式操作视角。

viewer (object) 视图对象
