new XLGlobe.Analysis.analysisBuffer(options)

缓冲区分析（坐标）。

options (Object?)
Name Description
options.viewer
(HTMLElement| Object) 		当前new XLGlobe实例。
options.units
(HTMLElement| string) 		缓冲半径单位（meter，miles, kilometers, degrees, or radians）
Example 
            //实例化缓冲分析
            var buffer = new XLGlobe.Analysis.analysisBuffer({
                viewer: viewer,
                units: 'meter'
            });
            //coords坐标数组，r半径
            var coords=[120,34];
            var r=1000;
            buffer.initPointBuffer(coords, r);
