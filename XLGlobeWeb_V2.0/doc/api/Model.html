Model

Model对象表示页面上的模型。它公开允许您以编程方式更改模型的属性，并在用户与其交互时激发事件。

Instance Members
▸ Buildflowline(options)

建筑流动线特效。

Parameters
options (object) 添加建筑3DTiles的特效设置。 有效值包括： 'tileset' ， 'backgroundcolor' ， 'linespeed' ， 'linewidth'，'brightness','glowRange'

options
Name Description
options.tileset
<string>
建筑的3DTiles设置,默认为null。
options.backgroundcolor
<string>
背景颜色，默认为rgba(0, 127.5, 255 ,1)。
options.linespeed
<float>
线流动速度，默认为2.0。
options.linewidth
<float>
流动线宽度，默认为0.005。
options.brightless
<float>
亮度，默认为100。
options.glowRange
<float>
扩散范围，默认为360.0。

▸ tileMonomer(options)

options
Name Description
options.type
<string>
3dtiles与建筑面对照方式，property为按对应属性，geometry为按空间位置（点击位置时会有误差）,model为模型变色,coverage为覆盖面。
options.comparisonField
<Array>
type为property时对照的字段[3dtiles字段名,对照面数据字段名]。
options.leftClick
<boolean>

options.leftcallback

左击回调。

<boolean>
