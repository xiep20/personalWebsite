createMap

createMap 根据配置文件创建地图。

new XLGlobe.createMap(options)
options (Object?)
Name Description
options.url
(HTMLElement| string) 		配置文件的URL地址。
options.data
(HTMLElement| Object) 		地图配置信息，与url选一。
options.serverURL
(HTMLElement| string) 		可选项，用于替换之前的URL地址。
options.id
(HTMLElement| string) 		DOM上容器的ID。
options.success
(HTMLElement| Function) 		地图成功加载完成后执行
Example 
            
    XLGlobe.createMap({
        id: 'cesiumContainer',
        url: "../config/config.1.json",
        success: function (_Globe, gisdata) { //地图成功加载完成后执行
            
        }
      });
