## 快速开始

XLGlobe可以对地图的颜色进行设置。

## 如何使用

设置地图亮度透明度

``` javascript
layer.defaultAlpha = 1 //透明度 0~1
layer.defaultBrightness = 2 //亮度 1为界点
layer.defaultContrast = 1 //对比度  1为界点
```
