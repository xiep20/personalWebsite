## XLGlobe.MapV.getCenterByCityName(cityName) 从城市名获取城市坐标

[MapV](https://mapv.baidu.com/ "MapV") 图层

###### Parameters

|  Name |  Description |
| ------------ | ------------ |
|  cityName |  城市名称 |

#### 示例

``` javascript
    XLGlobe.MapV.getCenterByCityName('武汉')//{lng: 114.3162, lat: 30.581084}
```
