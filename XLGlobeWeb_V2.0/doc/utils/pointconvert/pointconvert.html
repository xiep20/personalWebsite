Utils

new XLGlobe.Utils.pointconvert()

提供了百度坐标（BD09）、国测局坐标（GCJ02）、WGS84坐标系、Web墨卡托 4类坐标之间的转换,传入参数 和 返回结果 均是数组：[经度,纬度]。

Instance Members
▸ jwd2mct([a1, a2])

经纬度坐标转墨卡托坐标。

▸ mct2jwd([[a1, a2])

墨卡托坐标转经纬度坐标。

▸ bd2gcj([a1, a2])

百度坐标转国测局坐标。

▸ gcj2bd([a1, a2])

国测局坐标转百度坐标。

▸ bd2wgs([a1, a2])

百度坐标转WGS84坐标。

▸ wgs2bd([a1, a2])

WGS84坐标转百度坐标。

▸ wgs2gcj([a1, a2])

WGS84坐标转国测局坐标。

▸ gcj2wgs([a1, a2])

国测局坐标转WGS84坐标。

Example 
              var mct = new XLGlobe.Utils.pointconvert().jwd2mct([a1, a2]);   //经纬度坐标转墨卡托坐标，a1代表x坐标，a2代表y坐标。
