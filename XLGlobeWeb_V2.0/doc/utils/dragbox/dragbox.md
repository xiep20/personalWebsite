##   new XLGlobe.Utils.DragBox(e,t) div可拖拽

###### Parameters


|  Name |  Description |
| ------------ | ------------ |
|  e |  可以点击拖动的元素 |
|  t |  拖动的整体元素 必须是position: absolute |

#### 示例

``` javascript
    var e = document.getElementById('rightbottom-title')
    var t = document.createElement('div');
    new XLGlobe.Utils.DragBox(e,t)
```
