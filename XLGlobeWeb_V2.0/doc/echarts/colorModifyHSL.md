
## XLGlobe.Echarts.colorModifyHSL(color, number) 通过指定color，生成新RGBA色值

###### Parameters

|  Name |  Description |
| ------------ | ------------ |
|  color |  16进制色值 |
|  number |  正整数 |

#### 示例

``` javascript
    var color = "#5A94DF"
    var number = 10
    var color = XLGlobe.Echarts.colorModifyHSL(color, number) // rgba(223,112,90,1)
```
