## 快速开始

测量工具可以为用户提供 空间距离、地表距离、地表面积、高度差、三角测量、方位角的量测。

## 如何使用

地图上的测量工具

``` javascript
        //创建测量工具
        var measure = new XLGlobe.Control.MeasureTool({
            viewer: viewer,
            terrainProvider: terrainProvider, //地形
            show: true,
            // target: 'measure',
            // classname:''
        });
```
